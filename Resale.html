👖 Branded Resale Bottoms Data Analysis¶

Welcome to this data exploration project focused on resale bottoms — including pants, jeans, shorts, and other branded lower-body apparel.

This project aims to:

  • 🧼 Clean and prepare raw product data
  • 📊 Analyze brand trends, pricing, and condition distributions
  • 📦 Uncover valuable insights for optimizing resale strategy

🔧 Tools Used¶

  • Python (pandas, numpy)
  • Seaborn & Matplotlib (visualization)
  • Jupyter Notebook
